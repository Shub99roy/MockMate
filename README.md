# 📌 Overview

MockMate is an intelligent interview preparation platform that creates personalized mock interviews using AI. Users can select their role, experience level, and technologies, then participate in realistic interview sessions with AI-generated questions.

After every interview, the platform analyzes responses and provides detailed feedback, helping users identify strengths and improve weak areas before real interviews.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/shub99roy/mockmate.git
```

Move into the project

```bash
cd mockmate
```

Install dependencies

```bash
npm install
```

Install frontend dependencies

```bash
cd client
npm install
```

Install backend dependencies

```bash
cd ../server
npm install
```

---

# 🔐 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGODB_URI=

JWT_SECRET=

AI_API_KEY=
```

---

# ▶️ Run Locally

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
